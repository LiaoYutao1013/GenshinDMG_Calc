# GenshinDMG_Calc
The features below are developing 
Goal:
A specialized MATLAB-based analytical tool designed to model and visualize the damage mechanics of Genshin Impact. This project translates the game's complex damage formulas—incorporating Base ATK, Multipliers, Elemental Mastery, Crit Ratios, and Resistance Shred—into interactive visual data.
​Key Features:
​Damage Scaling Analysis: Visualize how specific stats (e.g., ATK% vs. Crit DMG) affect total output using 2D/3D plots.
​Optimal Build Calculator: Uses MATLAB’s optimization capabilities to find the "sweet spot" for artifact stat distribution.
​Multi-Scenario Simulation: Compare damage curves across different weapons and team buffs (Bennet burst, Kazuha swirl, etc.) in real-time.
